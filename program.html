Program details

Technical training meets community problem-solving.

Students learn practical data skills while investigating real social crises: food insecurity, poverty, housing instability, and homelessness risk.

Core Skills

  • Data cleaning and preparation
  • Exploratory Data Analysis
  • Statistical thinking
  • Visualization and storytelling
  • GIS mapping by city, ZIP, and district
  • Research and survey methods

Tools Introduced

  • Python for processing and automation
  • R for statistical analysis
  • Excel / Google Sheets
  • Tableau-style dashboard thinking
  • GIS / mapping tools
  • Presentation design

Final Deliverables

  • Community data dashboard
  • Access or risk map
  • Research summary
  • Policy-style recommendation memo
  • Capstone presentation
  • Student portfolio package
